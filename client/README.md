# Sports Live App 🏏

A real-time sports scoring application built using:

- React + Vite
- Tailwind CSS
- Node.js
- Express
- WebSockets

## Features

- Live score updates
- Admin scoring panel
- Real-time commentary
- WebSocket communication
- React Router

## Pages

- `/` → Viewer Page
- `/admin` → Admin Panel

## Installation

### Client

```bash
cd client
npm install
npm run dev
